Markdown Synthax for beginners!
Markdown is a lightweight markup language that's easy to learn and widely used for creating formatted text documents. Here are some common formatting features:
  • Titles and Headings
    • To create headings in Markdown, use hash (#) symbols. The number of hash symbols determines the heading level.
    # Heading 1
    ## Heading 2
    ### Heading 3
  • Text Styling
    • Bold Text: Surround the text with double asterisks or double underscores **Bold Text** or __Bold Text__. Italic Text: Surround the text with single asterisks or single underscores *Italic Text* or _Italic Text_.
  • Unordered Lists:
    • * Item 1
    * Item 2
    * Item 3
  • Ordered Lists:
    • 1. First Item
    2. Second Item
    3. Third Item
  • Links
    • Create links with square brackets for the link text and parentheses for the URL: [OpenAI's Website](https://www.openai.com/)
  • Images
    • Insert images in a similar way to links, but use an exclamation mark ! in front of the square brackets: ![Alt Text](https://www.example.com/image.jpg)
That's it! You can now write your own Markdown text inluding your favorite book, movie or TV series.
